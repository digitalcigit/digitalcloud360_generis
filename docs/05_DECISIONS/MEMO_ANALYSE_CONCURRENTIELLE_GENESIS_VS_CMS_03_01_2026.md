# MEMO : Analyse Concurrentielle Genesis vs CMS Existants

## 1. Contexte

Ce document analyse le positionnement de Genesis AI par rapport aux solutions de création de sites web existantes, identifie nos axes de différenciation et formalise notre proposition de valeur unique.

---

## 2. Paysage Concurrentiel

### 2.1 CMS Traditionnels (WordPress, Joomla, Drupal)

| Aspect | Description |
|--------|-------------|
| **Forces** | Écosystème riche, flexibilité maximale, open-source, communauté massive |
| **Faiblesses** | Complexité technique, maintenance lourde, failles sécurité, plugins payants, courbe d'apprentissage élevée |
| **Cible** | Développeurs, agences, utilisateurs techniques |
| **Prix** | Gratuit + hébergement (5-50€/mois) + thèmes/plugins (50-500€) |

### 2.2 Website Builders (Wix, Squarespace, Webflow)

| Aspect | Description |
|--------|-------------|
| **Forces** | Interface WYSIWYG intuitive, templates professionnels, hébergement inclus |
| **Faiblesses** | Coûteux à l'échelle, vendor lock-in, personnalisation limitée, export impossible |
| **Cible** | PME, freelances, créatifs |
| **Prix** | 12-45€/mois |

### 2.3 Headless CMS (Contentful, Strapi, Sanity)

| Aspect | Description |
|--------|-------------|
| **Forces** | API-first, performance optimale, découplage frontend/backend |
| **Faiblesses** | Nécessite développeur frontend, complexité technique, coût élevé |
| **Cible** | Équipes tech, startups, entreprises |
| **Prix** | 0-500€/mois selon usage |

### 2.4 No-Code Builders (Framer, Carrd, Notion Sites)

| Aspect | Description |
|--------|-------------|
| **Forces** | Rapidité extrême, simplicité, prix bas |
| **Faiblesses** | Fonctionnalités limitées, design générique, pas professionnel |
| **Cible** | Side-projects, landing pages, MVPs |
| **Prix** | 0-19€/mois |

### 2.5 AI Website Builders (Durable, 10Web AI, Hostinger AI)

| Aspect | Description |
|--------|-------------|
| **Forces** | Génération en quelques secondes, zéro effort initial |
| **Faiblesses** | Résultats génériques, édition limitée, "one-shot" sans contexte |
| **Cible** | Utilisateurs pressés, tests rapides |
| **Prix** | 12-30€/mois |

## 4. Nos 5 Innovations Clés

### 4.1 Coaching Maïeutique (Business Brief Profond)

**Le problème des autres :**
- CMS : Template → Remplir des champs génériques
- AI Builders : "Create a restaurant website" → Résultat générique

**Notre solution :**
```
Coaching 5 étapes conversationnel
    ↓
Business Brief riche (Vision, Mission, Clientèle, Différenciation, Offre)
    ↓
L'IA comprend VRAIMENT le business
    ↓
Site personnalisé + contenu contextuel pertinent
```

**Valeur :** Le site reflète l'identité unique du business, pas un template rempli.

---

### 4.2 Édition Conversationnelle Continue

**Le problème des autres :**
- WordPress : Apprendre Gutenberg/Elementor → Modifier manuellement
- Wix : Drag & Drop complexe → Modifier manuellement
- AI Builders : Génération unique, édition traditionnelle ensuite

**Notre solution :**
```
Utilisateur: "Ajoute une section témoignages avec 3 avis"
Genesis AI: → Génère la section avec contenu adapté au secteur
            → Intègre au bon endroit dans le site
            → Preview instantané

Utilisateur: "Les couleurs sont trop froides, réchauffe ça"
Genesis AI: → Ajuste la palette (primary, accent)
            → Applique cohérence sur tout le site
```

**Valeur :** L'utilisateur parle à son site comme à un assistant humain.

---

### 4.3 Site Évolutif & Proactif

**Le problème des autres :**
- Sites statiques créés une fois → oubliés → obsolètes
- Mises à jour = charge mentale pour l'entrepreneur

**Notre solution :**
```
Genesis AI (proactif):
├── "La Saint-Valentin approche, voulez-vous un menu spécial sur le hero?"
├── "Vos concurrents ont ajouté des avis Google, activer ce module?"
├── "Votre section 'À propos' n'a pas changé depuis 6 mois, actualiser?"
└── "Nouvelle tendance SEO détectée, optimiser vos titres?"
```

**Valeur :** Le site devient un partenaire qui anticipe les besoins.

---

### 4.4 Localisation Marché Africain Native

**Le problème des autres :**
- Prix en USD/EUR inadaptés au pouvoir d'achat
- Paiements Stripe/PayPal uniquement (faible pénétration en Afrique)
- Templates et exemples occidentaux
- Serveurs distants = latence

**Notre solution :**
```
Genesis Africa-First:
├── Prix en FCFA / monnaies locales
├── Mobile Money natif (Orange Money, MTN, Wave)
├── Templates secteurs locaux (maquis, salon coiffure, transport)
├── Support français + langues locales
└── CDN optimisé pour l'Afrique
```

**Valeur :** Première solution de création de sites vraiment conçue pour l'Afrique.

---
